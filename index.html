About the project

The goal of this project was to show the UI/UX process for the game Overwatch 2. This project started with the player journey and ended with UI Mockups and a Colorblindness test.

Roles and Responsibilities

  • The role in this project was to recreate the UI/UX design for Overwatch.
  • Some responsibilities included designing Wireframes, Usability Tests and Player Profiles.
  • Create a UI styleguide along with UI Mockups for the game.
  • Design Wireframes & assets that match the style of the game.

Length of the Project

8 weeks

Challenges

  • Making changes that fit the needs of all players.
  • Recreating styles and assets.

Usability Test Results

  • After navigating through the Character Screen, players found the arrows to be too much work when selecting their character. They preferred to see all the characters faces as a whole.
  • Iterated Wireframe shows the Character Screen without arrows.
  • Iterated Wireframes also showcases highlighted character boxes so players can easily see each character..
  • A hover effect was added.
  • The character description was not necessary when choosing a character and was removed.

Outcome

  • After using the color blindness simulator and upon assessing each filtered mockup, no iterations were needed.
  • The contrast is heightened to create more clarity for all players.
  • The font is large and clear so that any player is able to see the buttons and what they need to do next in the game.
  • The color choices are on the brighter side.
